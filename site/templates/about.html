About

Created by Dr. Qiang Hao and Nathan Tresham. Department of Computer Science, College of Science and Engineering, Western Washington University. All source code is available on GitHub here.

The goal of this project is to create a service that can spawn and manage bots that provide answers to previously-answered questions on a Piazza forum using machine learning and natural language processing to find the intent of a given question and determine whether two questions are asking the same thing.

More specific goals of the project are as follows: the project should be able to...

  • pull, clean, and store data from Piazza in a database for better performance,
  • present data in the form of a web-based application,
  • and render question and answer data in a different way than on Piazza; e.g., with the help of visualizations--collapsable questions with follow-ups/answers, differentiate learning questions from non-learning questions, etc.
